# Placement AI 🚀

An AI-powered web application that predicts student placement probability and provides personalized improvement insights.

---

## 🌐 Live Demo
👉 https://placement-ai-jewd4yckezrnbxfhbfn3ep.streamlit.app/

---

## 📌 Overview

Most students don’t know where they stand in terms of placement readiness.

This project solves that by:
- Predicting placement probability using a Machine Learning model
- Identifying weak areas in a student's profile
- Providing actionable, AI-style recommendations for improvement

---

## ⚙️ Features

- 📊 Placement probability prediction
- 🧠 Context-aware insights (not generic advice)
- 🎯 Identification of weakest areas
- 🔄 Real-time updates based on user input
- 🌐 Fully deployed interactive web app

---

## 🧠 How it Works

1. User inputs:
   - CGPA  
   - Projects  
   - Internships  
   - Certifications  
   - Backlogs  

2. Model predicts placement probability using a trained **Random Forest Classifier**

3. A reasoning layer:
   - Analyzes feature impact  
   - Filters unrealistic suggestions  
   - Generates personalized recommendations
